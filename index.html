Entendendo Comunicação Client x Server

História da Web

  1. Apresentação do Curso e Professor
    2.

    Introdução ao conteúdo programático do curso e ao perfil do professor, estabelecendo as bases para o aprendizado e a interação.

  2. Primeiros Computadores
    3.

    Máquinas como o ENIAC e UNIVAC, desenvolvidas em meados do século XX, eram gigantescas e destinadas a cálculos complexos, marcando o início da era da computação.

  3. Como Surgiu
    4.

    A comunicação Client-Server surgiu da necessidade de centralizar recursos e dados, permitindo que múltiplos usuários acessassem as mesmas informações e serviços de forma distribuída.

  4. Grandes nomes
    5.

    Tim Berners-Lee (criador da WWW), Bill Gates (criadiador da Microsoft) e Steve Jobs (criador da Apple) são figuras centrais na arquitetura da internet e da comunicação Client-Server.

O que são Clients

  1. Client no Linguajar Web
    2.

    No contexto da web, o cliente é tipicamente o navegador (browser) que o usuário utiliza para acessar websites. Ele envia requisições HTTP e renderiza o conteúdo recebido.

  2. Navegadores
    3.

    Softwares como Chrome, Firefox, Edge e Safari que interpretam HTML, CSS e JavaScript para exibir páginas da web e interagir com servidores.

  3. Aplicações Web
    4.

    Softwares acessados via navegador, sem necessidade de instalação local. Exemplos incluem Gmail, Facebook e Google Docs, que funcionam com a interação entre cliente e servidor.

  4. Dispositivos Móveis
    5.

    Smartphones e tablets que, através de navegadores ou aplicativos específicos, atuam como clientes para acessar serviços e conteúdos na internet, adaptando a experiência para telas menores.

O que são Servers

  1. Sobre Servidores
    2.

    Servidores são máquinas potentes, otimizadas para processar múltiplas requisições simultaneamente, armazenar grandes volumes de dados e garantir a disponibilidade de serviços.

  2. Hardware e Softwares em Servidores
    3.

    Hardware inclui processadores robustos, muita RAM e armazenamento rápido; software abrange sistemas operacionais (Linux, Windows Server), servidores web (Apache, Nginx) e bancos de dados.

  3. Tipos de Servidores
    4.

    Existem diversos tipos, como servidores web (para sites), servidores de banco de dados (para dados), servidores de e-mail, servidores de arquivos e servidores de aplicativos, cada um com uma função específica.

  4. Hospedagem de Sites
    5.

    Serviço que permite que um site seja armazenado em um servidor web e acessível publicamente pela internet, garantindo que ele esteja online 24/7.

Linguagens de Programação

  1. O que é
    2.

    Refere-se à lógica de programação que é executada no servidor. Linguagens como JavaScript (Node.js), Python, PHP e Java são usadas para processar dados, interagir com bancos e gerar conteúdo dinâmico.

  2. Server-Side
    3.

    Refere-se à lógica de programação que é executada diretamente no navegador do usuário. JavaScript, HTML e CSS são as principais tecnologias para criar interfaces interativas e dinâmicas no lado do cliente.

  3. Client-Side
    4.

    Refere-se à lógica de programação que é executada diretamente no navegador do usuário. JavaScript, HTML e CSS são as principais tecnologias para criar interfaces interativas e dinâmicas no lado do cliente.

  4. HTML não é Linguagem de Programação
    5.

    HTML (HyperText Markup Language) é uma linguagem de marcação usada para estruturar o conteúdo de uma página web, definindo elementos como parágrafos, títulos e imagens, mas não possui lógica de programação.

Introdução ao HTML na Prática

  1. Aprensentando o Curso
    2.

    Visão geral dos objetivos de aprendizagem, metodologia e estrutura do curso, preparando o aluno para o que será abordado.

  2. Ferramentas Utilizadas
    3.

    Editor de texto (VS Code) navegador web (Chrome) e o próprio sistema operacional para desenvolver e testar o código HTML.

  3. Inspetor de Elementos
    4.

    Ferramenta integrada aos navegadores que permite visualizar e modificar o HTML, CSS e JavaScript de uma página em tempo real, útil para depuração e aprendizado.

  4. Estrutura Básica do HTML
    5.

    Inclui metadados, título e conteúdo visível da página, formando o esqueleto de qualquer documento web.

  5. Falando sobre Tags
    6.

    Tags HTML são palavras-chave entre que marcam o início e fim de um elemento, definindo sua função e como o navegador deve interpretá-lo.

  6. Atributos das Tags
    7.

    Propriedades adicionais dentro de uma tag de abertura que fornecem informações extras ou modificam o comportamento do elemento.

  7. Textos
    8.

    Elementos HTML usados para estruturar e formatar o conteúdo textual em uma página.

  8. Listas Ordenadas e Não Ordenadas
    9.

    Ordered list para listas numeradas e (unordered list) para listas com marcadores, usando (list item) para cada item da lista.

  9. Links
    10.

    Criados com a tag (anchor) e o atributo href (hypertext reference), que especifica o URL de destino, permitindo a navegação entre páginas web.
